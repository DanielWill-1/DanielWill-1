## Featured Projects

### Adaptive Few-Shot Learning Defect Detection System

A meta-learning system for steel surface defect detection using Prototypical Networks, spatial attention, episodic training, and prototype refinement. The project focuses on few-shot classification, Grad-CAM interpretability, t-SNE embedding analysis, and multi-seed evaluation.

### Mamba-UNet for Diabetic Foot Ulcer Segmentation

A hybrid medical image segmentation project combining UNet-style dense skip connections with state-space modeling ideas for long-range spatial dependency learning. The project explores deep supervision, augmentation, and boundary-aware evaluation for diabetic foot ulcer segmentation.

### End-to-End Diabetes Prediction Pipeline

An applied MLOps project covering the machine learning lifecycle, including preprocessing, model training, experiment organization, CI/CD, infrastructure-as-code concepts, and automated deployment workflows.

### Auto Comic Translator

An OCR and translation pipeline for webcomic panels. The project combines text detection, translation, and image processing to support automated multilingual comic reading workflows.

### Finite Automata Simulator

A visual DFA/NFA simulator for learning automata theory through state transitions, path tracing, and graph-based visualization. Built with Flask, React, and Graphviz.

### Spotify Music Recommender System

A machine learning recommender system using audio features, content-based filtering, K-Means clustering, PCA-based similarity, and a hybrid popularity-aware recommendation strategy.
